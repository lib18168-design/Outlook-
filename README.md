# Outlook 账号每日限额管理工具

管理 Outlook 账号的每日使用次数（每个账号每天5次），支持批量导入、自动轮换、导出报表等功能。

## 功能特性

- 账号管理（增删改查、分组）
- 每日使用次数限制（每个账号每天5次）
- 自动轮换可用账号
- 批量导入/导出 CSV
- 统计面板（总账号、可用数量、今日已用次数等）

## 技术栈

- Flask + SQLAlchemy
- SQLite
- Pandas（导入导出）
